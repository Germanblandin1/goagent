# goagent/mcp

MCP (Model Context Protocol) integration for [goagent](https://github.com/Germanblandin1/goagent).

Wraps [mark3labs/mcp-go](https://github.com/mark3labs/mcp-go) so that MCP tools are indistinguishable from local tools in the ReAct loop. Also exposes a lightweight MCP server builder so goagent tools can be served to other MCP clients.

```bash
go get github.com/Germanblandin1/goagent/mcp
```
