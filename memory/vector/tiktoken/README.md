# goagent/memory/vector/tiktoken

Tiktoken-backed `SizeEstimator` for [goagent](https://github.com/Germanblandin1/goagent).

Provides exact token counts for OpenAI models using the real tiktoken tokenizer. Kept as a separate module so the core `memory/vector` package has zero external dependencies.

```bash
go get github.com/Germanblandin1/goagent/memory/vector/tiktoken
```
