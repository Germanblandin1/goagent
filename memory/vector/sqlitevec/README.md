# goagent/memory/vector/sqlitevec

SQLite + sqlite-vec store for [goagent](https://github.com/Germanblandin1/goagent).

Implements `goagent.VectorStore` over SQLite with the [sqlite-vec](https://github.com/asg017/sqlite-vec) extension. Suitable for embedded and single-process deployments with no external database dependency.

```bash
go get github.com/Germanblandin1/goagent/memory/vector/sqlitevec
```

> **Requires CGO and a C compiler.**
