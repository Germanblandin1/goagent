# goagent/providers/ollama

Ollama provider for [goagent](https://github.com/Germanblandin1/goagent).

Implements `goagent.Provider` and `goagent.Embedder` targeting a locally running Ollama instance via its OpenAI-compatible API. Useful for local development and offline agents.

```bash
go get github.com/Germanblandin1/goagent/providers/ollama
```

Requires [Ollama](https://ollama.com) installed and running locally.
