# goagent/providers/anthropic

Anthropic provider for [goagent](https://github.com/Germanblandin1/goagent).

Implements `goagent.Provider` backed by the Anthropic Messages API (Claude). Supports streaming, extended thinking, and prompt caching.

```bash
go get github.com/Germanblandin1/goagent/providers/anthropic
```

Requires the `ANTHROPIC_API_KEY` environment variable or `WithAPIKey` option.
