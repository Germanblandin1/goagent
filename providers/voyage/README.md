# goagent/providers/voyage

Voyage AI embedder for [goagent](https://github.com/Germanblandin1/goagent).

Implements `goagent.Embedder` backed by the Voyage AI embeddings API — the embedding service recommended by Anthropic for use with Claude.

```bash
go get github.com/Germanblandin1/goagent/providers/voyage
```

Requires the `VOYAGE_API_KEY` environment variable or `WithAPIKey` option.
