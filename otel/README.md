# goagent/otel

OpenTelemetry integration for [goagent](https://github.com/Germanblandin1/goagent).

Translates goagent hook events into OpenTelemetry spans and metrics following the RED pattern (Rate, Errors, Duration). Drop-in with any OTel-compatible backend (Jaeger, Prometheus, Honeycomb, etc.).

```bash
go get github.com/Germanblandin1/goagent/otel
```
